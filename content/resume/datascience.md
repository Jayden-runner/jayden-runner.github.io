## 项目经历

### 高维系统稳定性分析与关键驱动因子识别 | 负责人 | 2026.03 — 至今

> 量化高维系统（2 万—5 万特征维度）的稳定性指标，识别关键驱动因子并验证统计显著性

**高维数据降维与特征工程**：针对原始数据矩阵（维度 1.28×10⁷）的 SVD 计算瓶颈，开发随机 SVD 近似算法，将计算复杂度降低至 1.1×10⁶（降幅约 10—12 倍），保留 99%+ 方差信息，支撑后续分析

**业务指标相关性分析**：系统扫描 16 种参数组合下的稳定性指标，通过 Pearson 相关性检验量化参数影响：特征维度数与稳定性指标显著正相关（r=0.63, p<0.01）；层级复杂度与频域指标显著负相关（r=-0.60, p<0.05），为参数优化提供统计依据

**技术栈**：Python / NumPy / SciPy / Matplotlib / Seaborn / Scikit-learn / 随机 SVD / 假设检验

### 大规模图像数据处理与质量评估管道 | 负责人 | 2025.03 — 2026.03

> 构建大规模图像数据处理与质量评估管道，从 9 万+ 图像中筛选出 1 万+ 高质量样本，测试集 MAE 降至 0.13—0.16

**多维度数据清洗与筛选**：使用 Python 批量处理 90,000+ CelebA 人脸图像，设计并执行基于结构方差、数值稳定性的多维度数据清洗与筛选逻辑，最终保留 10,240 张高质量样本

**质量评估体系搭建**：建立测试集 MAE（0.13—0.16）和 MSE Loss 作为核心质量评估指标，持续监控模型输出还原度，确保五官位置、发型等关键特征的有效保留

**结构化质量报告**：通过像素级对比分析，量化评估解码后图像中轮廓、五官位置、发型等特征的还原效果，形成结构化质量报告

**技术栈**：Python / NumPy / Pandas / Matplotlib / Scikit-learn

### 统计建模与随机模拟 | 负责人 | 2024.09 — 2026.01

> 应用统计建模与随机模拟方法，完成多个物理系统数据集的特征提取与模式分析

**大规模随机抽样与统计推断**：实现 10 万样本蒙特卡洛积分，通过随机采样与误差控制，将计算精度稳定提升至 10⁻³ 量级；进一步基于计盒维数法对多组时间序列进行复杂度特征量化分析，识别出不同动态模式

**参数空间网格扫描与相关性分析**：对反应扩散模型进行 1000×1000 参数网格扫描，采集超过 10⁶ 个样本点，通过可视化分岔图揭示系统状态随参数演化的关键拐点，发现多个相变区域

**高维时间序列特征提取**：基于 FPUT 模型构建 N 粒子链，使用 RK4 积分器生成 10⁶ 步长时间序列，采用正弦模态分解提取能量谱特征，验证了多种模态成分的周期性回归规律

**技术栈**：Python / NumPy / SciPy / Matplotlib / Pandas / 时间序列分析 / 统计推断
