## 项目经历

### 神经网络学习动力学的混沌性质的研究 | 负责人 | 2026.03 — 至今

> 量化 2 万—5 万参数 MLP 的混沌动力学特性，揭示架构参数与频域稳定性的显著相关性

**高维 Jacobian 计算与 SVD 降维**：基于 PyTorch 自动微分，对 2 万—5 万参数规模的 MLP 实现精确 Jacobian 计算；针对精确 SVD 算力瓶颈（典型场景 1.28×10⁷ flops），开发随机 SVD 近似算法，k=10 时复杂度降至 1.1×10⁶ flops，降低约 10—12 倍

**系统参数扫描与统计分析**：对 16 种 MLP 架构进行全参数扫描，通过相关性分析量化架构与稳定性关系：每层神经元数与频域稳定性显著正相关（r=0.63, p<0.01）；隐藏层数与频域 FTLE 显著负相关（r=-0.60, p<0.05）

**技术栈**：Python / PyTorch / CUDA / NumPy / SciPy / Matplotlib / Seaborn / Scikit-learn

**量化评估**：测试集 MAE 0.13—0.16，MSE 损失稳定收敛；解码后人脸轮廓、五官位置、发型特征均有效还原，验证框架端到端可行性
